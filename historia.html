HISTORIA DE HTML

El nacimiento de una web: A principios de los 80, el físico Tim Berners-Lee, trabajando en el CERN, concebía una nueva forma de compartir información: el hipertexto. Esta idea revolucionaria daría lugar a HTML(). En 1991, se publicó formalmente el primer documento que describía HTML, marcando el inicio de lo que hoy conocemos como la World Wide Web. Este lenguaje de marcado, simple pero poderoso, permitía estructurar contenido en forma de encabezados, párrafos, listas y enlaces. Con HTML(), la información podía ser conectada de manera no lineal, creando una red de documentos interconectados. Berners-Lee imaginó una web en la que cualquier persona pudiera crear y compartir información, democratizando el acceso al conocimiento.

La estandarización y la expansión: A medida que la web ganaba popularidad, la necesidad de un estándar para HTML se hizo evidente. En 1995, el W3C (Consorcio World Wide Web) asumió la responsabilidad de definir y mantener los estándares de HTML. Con la publicación de HTML 2.0, se estableció un marco sólido para el desarrollo web. Las versiones posteriores introdujeron nuevas características como tablas, formularios y soporte para multimedia. Sin embargo, la falta de una implementación consistente entre los navegadores generó una fragmentación en la web. Para abordar este problema, se desarrolló XHTML, una versión más estricta de HTML basada en XML. XHTML buscaba imponer reglas más rigurosas para la sintaxis, con el objetivo de facilitar la validación y mejorar la interoperabilidad entre diferentes navegadores.

En diseño, aparte de cumplir con las especificaciones propias del lenguaje, debe respetar ciertos criterios de Accesibilidad Web, siguiendo unas pautas o las normativas y leyes vigentes en los países donde se regule dicho concepto. Este se encuentra disponible y desarrollado por el W3C a través de las Pautas de Accesibilidad al Contenido Web 1.0 WAG (actualizadas recientemente con la especificación 2.0).

HTML5: El futuro de la web HTML5 representó un gran avance en el desarrollo web. Publicado en 2014, este estándar unificó y amplió las capacidades de HTML, CSS y JavaScript. HTML5 introdujo nuevas etiquetas semánticas para mejorar la accesibilidad y el SEO, así como características multimedia integradas. Además, HTML5 facilitó la creación de aplicaciones web ricas y interactivas, reduciendo la necesidad de plugins externos. Con HTML5, la web se convirtió en una plataforma más poderosa y versátil para el desarrollo de aplicaciones. Las aplicaciones web progresivas (PWA) surgieron como una consecuencia directa de HTML5, permitiendo a las aplicaciones web funcionar de manera similar a las aplicaciones nativas en dispositivos móviles.
